
# Сайт с информацией о последствиях и способах борьбы с глобальным потеплением
## Иедя для проекта:
> Сайт должен содержать информацию о глобальном потеплении, последствиях и способы борьбы с ним
> Сайт должен содержать ссылки на полезные ресурсы и актуальную информацию о глобальном потеплении
> Сайт должен содержать авторизацию пользователя
> Сайт содержит функцию написание комментариев и их историю
## Технологии:
> Flask
> SqlAlphemia
> html и css
## База данных:
### Таблица пользователя
> id intager primary key
> username string(50) not null
> email string(100) not null
> password
### Таблица комментариев:
> id
> comment
> time
> username
